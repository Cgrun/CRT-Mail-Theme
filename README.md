# CRT-Mail-Theme
a retro amber terminal theme for Outlook Web, inspired by old curved-glass CRT displays and classic phosphor monitors. It transforms the mail interface into a calm, dark archive-like workspace with warm amber text, subtle scanlines, soft glow, and a vintage terminal atmosphere while keeping readability and usability first.

Also can be used on Youtube.
